--------------------------------
快捷键
--------------------------------

1.填充前景色到画布
alt+del

2.切换前景色/背景色
x

3.反向
ctrl + i

4.拷贝图层
ctrl + j

5.替换颜色
选中图层后
alt + i + j + r

6.保存
ctrl + s

7.另存为
alt + f + a
shift + ctrl + s

8.调节色相
ctrl + u

9.建立辅助线
ctrl + r 打开标尺
在标尺的刻度线上拉

10.标准线
ctrl + ；
